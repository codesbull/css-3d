# css-3d

#### 介绍

#### 安装教程

1.  下载源码
2.  修改img文件夹下的图片（图片名称不变）
3.  双击index.html即可显示3d相册效果

#### 使用说明

1.  css中可以修改背景颜色
2.  音乐播放需要编译工具运行，双击触发不了
（可以在audio标签加上controls属性，显示播放条）
<audio loop src="img/qianyuqianxun.mp3" id="audio" controls autoplay preload="auto">
